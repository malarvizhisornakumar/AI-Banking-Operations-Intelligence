# AI Banking Operations Intelligence

## Overview

This project demonstrates a production-ready Banking Operations Analytics platform built using SQL Server, Python, Power BI, and AI.

The goal is to simulate a real banking analytics solution used for monitoring operational KPIs, SLA performance, OCR processing, productivity, and data quality.

## Tech Stack

- SQL Server
- Power BI
- Python
- ETL
- AI
- GitHub
